Центрирование
Общая функция CSS — центрирование текста или изображения. Фактически, существует три вида центрирования: Центрирование строк текста Центрирование текстового блока или изображения Центрирование текстового блока или изображения по вертикали В последних реализациях CSS вы можете также использовать возможности Уровня 3 (CSS3), позволяющие выровнять по центру абсолютно позиционированные элементы: Выравнивание по вертикали в уровне 3 Выравнивание по вертикали и горизонтали в уровне 3 Выравнивание в области просмотра в уровне 3
